- TypeScript
- Next.js
- MUI
- ethers.js
- EVM based Smart Contracts
- The Graph
- Zora (https://zora.co/)
- Lit Protocol (https://litprotocol.com/)
- Vercel

## Commands

- Install depencies - `npm install`
- Start development server - `npm run dev`
- Deploy to Vercel - `vercel`
- Deploy to production on Vercel - `vercel --prod`
